# Snake (ncurses) with multithreading

## Описание
Учебная реализация игры Snake на C++ с использованием библиотеки ncurses.
В рамках задания выполнен рефакторинг исходного примера и добавлена вторая змейка,
работающая в отдельном потоке с использованием std::thread.

Проект демонстрирует основы многопоточности и синхронизации потоков в C++.

## Функциональность
- Управление первой змейкой с клавиатуры (стрелки)
- Вторая змейка (автономная), работающая в отдельном потоке
- Одновременное движение двух змей
- Синхронизация потоков с помощью std::mutex
- Корректное завершение потоков
- Консольный интерфейс на ncurses

## Используемые технологии
- C++17
- std::thread
- std::mutex
- std::atomic
- ncurses
- Makefile
