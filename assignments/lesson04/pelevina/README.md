# Code Review AI Agent

## Идея
Агент предназначен для автоматического анализа исходного кода
(C++ / Python) и выдачи рекомендаций по улучшению качества кода.
Агент использует удалённый LLM API (Hurated) и реализован на базе
примера `ai_agent_example`.

## Возможности
- Анализ исходного кода из файла
- Поиск возможных ошибок
- Выявление плохих практик
- Рекомендации по улучшению и стилю

## Архитектура
Агент расширяет базовый класс `AiAgent` из примера:
- используется чтение `config.json` и `prompt.json`
- код из файла подставляется в prompt
- ответ LLM выводится в терминал
